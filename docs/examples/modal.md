# 上下文模态框示例
本例展示了如何使用 React Router 创建由路径决定的上下文模态框（2010年在 Pinterest、Instagram 流行）。特点是当你在同一父页面触发 URL 改变时，子页面会以模态框弹出显示；然而当你直接访问这个 URL 时，页面会直接显示完整的子页面而非弹出框。
