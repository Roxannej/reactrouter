# 基础示例
本例展示了 React Router 的部分基本特性，包括：       
- 布局和嵌套 `<Route>`
- 索引 `<Route>`
- 全匹配 `<Route>`
- 使用 `<Outlet>` 作为子路由的占位符
- 使用 `<Link>` 来导航
